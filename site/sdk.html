SDKs for drive9.

Build on drive9 directly with SDKs for Go, Python, TypeScript, and Rust. Use the SDK when you want to integrate drive9 into your own tools and services. If you want an agent to set itself up, start from the homepage instruction instead.

01

Supported languages

drive9 currently exposes SDKs for Go, Python, TypeScript, and Rust. They are meant for direct programmatic access from your own applications, automations, and internal tools.

02

When should I use the SDK?

Use the SDK when you want to build directly on drive9 as a developer. If you want an agent to install and configure drive9 for itself, use skill.md from the homepage instead of starting with the SDK.

03

Where do I start?

Start with the README for installation, authentication, and usage details. Full examples and language-specific packages belong in the docs and source repository, not on the landing page.
